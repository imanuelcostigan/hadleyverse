hadleyverse
===========

A package to install packages from the hadleyverse

## List of packages

The list of packages that will be installed by default include:

* ggplot2
* lubridate
* reshape2
* stringr
* plyr

## How to

Install the packages:

```
install_the_hadley_verse()
```

Load the packages:

```
load_the_hadley_verse()
```
